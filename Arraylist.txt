Arraylists
good for multithreading

Java has 2 implementations of dynamic arrays
Vector : grows by 100% of its size once it gets full - All methods in vector is synchronized(only a single thread can access that method)

ArrayList: grows by 50% of its size once it gets full


declare an arraylist
ArrayList is a dynamic array

ArrayList<Integer> list = new ArrayList<>();
list.add(10);
list.add(20);
list.add(30);
list.remove(index:0);
list.indexOf(20); <--returns index of element 20
list.lastIndexOf(o:20); <-- will retrieve the index of the last occurence of 20
list.contains(20);<--returns boolean value if we have this item in our array or not
list.size();<-- gets the number of items in array
list.toArray();<-will turn into a regular array object
System.out.println(list);

wil print out:
[20,30]
